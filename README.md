## Installation

> [!NOTE]

> | Library | Version |
> | --- | --- |
> | ffmpeg | 4.4.2 |
> 
> For information, the code has been developed and works on PowerShell with the following library versions:
> | Library | Version |
> | --- | --- |
> | ffmpeg | 2024-03-07 |

> | Library | Version |
